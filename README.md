# SeedTableV3
Seed Table v3 mod for Conan Exiles Enhanced

Putting this here as I don't have the time to update it anymore, and maybe somebody will want it in the future.

A few notes to any modders or future modders looking to use this:

- I removed all of the AoC and Emberlight-related images. They gave me permission to use them, but not the general public. All of the data tables and references are still intact otherwise. Side note: Do NOT go spamming the devs of these mods asking for permission to use their assets. Sorry, not sorry. Technically, this is all broken until they update anyways.

- This is by no means a "perfect" project. I did things the way I knew how to. They might not be the most efficient or cleanest way. I did my best, though, to go through and at least comment on what stuff does where relevant.

- Yes, there are like... 3 or 4 different checks in place for the cross-mod integration. I swear mod users will find a way for it to not work.

- This probably only works for the UE5 devkit.

That's all. Go nuts. Or don't.
